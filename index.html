Usage

In most cases you want to use the concatenated and minified generated by the minify_css.sh script, as it is smallest.

Minified version: style.min.css

<link rel="stylesheet" href="style.min.css">

Concatenated version(human-readable): style.css

<link rel="stylesheet" href="style.css">

Include all CSS files manually(does not require minify_css.sh):

<link rel="stylesheet" href="base.css">
<link rel="stylesheet" href="utils.css">
<link rel="stylesheet" href="grid.css">
<link rel="stylesheet" href="page.css">
<link rel="stylesheet" href="button.css">
<link rel="stylesheet" href="table.css">
<link rel="stylesheet" href="toolbar.css">

A gzipped version is also provided for webservers that can't automatically gzip.

For development purposes, there is the start_dev.sh script that uses the busybox httpd and inotifywait for hot reload.

Buttons and button groups

The btn classes can be used to create buttons.

Every button needs this class, all other classes are optional.

Button colors

There are 4 color variants:

btn-blue and btn-red

Button sizes

btn-large will make a larger button.

btn-small will make a smaller button. Ideal for inline use.

Block buttons

Block buttons take up all of the available width.

Round buttons

You can make a button round by adding the round class.

Button groups

Using button groups you can indicate that a group of buttons belongs together.

They will be right next to each other vertically on wide displays, and horizontally on smaller displays.

The border lines of adjacent buttons will merge.

Grid Layout

The grid class provides a single row of items on desktop(width>768px).

On wide displays the items will be placed next to each other vertically.

On smaller displays(<768px) they will flow from top to bottom(has no effect).

You can use the classes col-1, col-2, col-3, col-4, col-5, col-6 to set the "width" in the container.
They're simply a shortcut for setting flex-grow.

Code blocks

You can use the codeblock class on a codeblock or pre tag to get proper formatting for code. 

Use a pre tag to create a full width code block,
or a code tag to create an inline code block.

Tables

Styling HTML tables.
